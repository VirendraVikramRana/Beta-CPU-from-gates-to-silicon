# Beta-CPU-from-gates-to-silicon
Building a complete Beta CPU from gates to silicon — MIT 6.004 Spring 2017
Self-study of MIT OCW 6.004 (Spring 2017) — building a complete 
32-bit Beta CPU from CMOS transistors up, using JADE and Verilog

## Tools Used
- JADE (MIT 6.004 circuit simulator)
- Verilog / EDAPlayground
- Xilinx Vivado (synthesis & simulation)
